Troubleshooting Guide

Common Issues and Solutions
Issue: Cannot access camera for QR login

Solution: Make sure you've granted camera permissions in your browser settings. Try refreshing the page and allowing camera access when prompted.

Issue: System call data not updating

Solution: Check your internet connection and refresh the page. The data updates automatically every 5 seconds by default.

Issue: Activity logs not showing

Solution: Make sure you're logged in and have performed some actions. Use the Auto Fetch button to refresh the logs.

Issue: Performance metrics seem incorrect

Solution: System call monitoring requires proper permissions. Make sure the application has the necessary system access.

Tips & Best Practices

  • Keep your QR code secure and don't share it with others
  • Regularly review your activity logs for security
  • Use filters in the Optimizer to focus on specific system calls
  • Export reports regularly for record keeping
  • Check the documentation for advanced features
  • Use the Auto Fetch feature for real-time monitoring
