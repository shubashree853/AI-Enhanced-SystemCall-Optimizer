Documentation

Getting Started

1. Registration

Create your account by clicking the "Register" button. Fill in your details including username, email, and password. A unique QR code will be automatically generated for your account.

2. Login

You can login using:

  • Traditional Login: Username and password
  • QR Code Login: Scan your QR code using the camera
3. Dashboard

After logging in, you'll see your dashboard with:

  • User information and role
  • Your QR code for quick login
  • Quick stats and activity logs
  • Performance trends

System Call Optimizer

Overview

The System Call Optimizer provides real-time monitoring and analysis of system calls with AI-powered optimization recommendations.

Key Features
  • Real-time Monitoring: View system calls as they happen with live updates
  • Performance Metrics: Track execution time, CPU usage, memory, and I/O
  • Category Filtering: Filter by File I/O, Process, Memory, IPC, etc.
  • AI Recommendations: Get intelligent suggestions for optimization
Using the Optimizer
  1. Navigate to the Optimizer page from the header
  2. Use filters to focus on specific system calls
  3. Click on any system call to view detailed information
  4. Review AI recommendations for performance improvements

QR Code Authentication

How It Works

Each user receives a unique QR code upon registration. This QR code contains a secure token that can be used for quick authentication.

Using QR Login
  1. Go to the QR Login page
  2. Allow camera access when prompted
  3. Point your camera at your QR code
  4. You'll be automatically logged in when the code is detected
QR Code Management
  • Regenerate: Create a new QR code (invalidates the old one)
  • Revoke: Temporarily disable your QR code
  • Activate: Re-enable a revoked QR code

Activity Logs & Reports

Activity Logs

Track all your activities including logins, QR code operations, and system interactions. Use filters to find specific activities by date, action type, or other criteria.

Auto-Fetch Feature

Click the "Auto Fetch" button to automatically refresh activity logs every 5 seconds. This is useful for monitoring real-time activities.

Reports

Generate comprehensive reports with:

  • Activity statistics and trends
  • Visual charts and graphs
  • Export to CSV format

API Documentation

Available Endpoints
  • GET /optimizer/performance/ - Get performance data
  • GET /optimizer/recommendations/ - Get optimization recommendations
  • GET /optimizer/categories/ - Get syscall categories
  • POST /api/qr-login/ - Login using QR code token
  • GET /api/dashboard-stats/ - Get dashboard statistics
Authentication

Most endpoints require authentication. Include your session cookie or use QR code token authentication for API access.
